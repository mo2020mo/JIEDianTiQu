dir 命令展示当前目录下的文件和目录

cd 目录名称进入到某个目录中 比如，运行dir命令的结果，显示当前目录下有一名字叫go的文件夹，你可以使用 cd go 进入go文件夹

安装cloudflare c3工具并创建项目:
npm create cloudflare@latest
安装wrangler工具
npm install -g wrangler
测试程序
npx wrangler dev
安装js-yaml的依赖
npm install js-yaml
部署程序到cloudflare worker
npx wrangler deploy
